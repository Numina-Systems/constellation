# Human Test Plan: GH-23 Hybrid Search + Multi-Domain Retrieval

## Prerequisites

- PostgreSQL 17 with pgvector running (`docker compose up -d`)
- Migrations applied: `bun run migrate`
- Configured embedding provider (Ollama at `192.168.1.6:11434` or configured alternative)
- `config.toml` with valid embedding provider settings
- All automated tests passing: `bun test`
- Build compiles cleanly: `bun run build`

## Phase 2: Search Tool Smoke Test via REPL

| Step | Action | Expected |
|------|--------|----------|
| 2.1 | Start the daemon: `bun run start` | REPL starts without errors; "search tools registered" in output |

| 2.5 | Send: "Search for programming using semantic mode" | Agent dispatches `search` with `mode: 'semantic'`. Results based on embedding similarity. |
